# go-web-sample

Go で作成されたシンプルな認証付き Web アプリケーションサンプルです。セッション管理、ログイン機能、ミドルウェアを使用した認証処理を実装しています。

## 概要

このプロジェクトは、Go の標準ライブラリと`gorilla/sessions`を使用した、基本的な Web アプリケーションのテンプレートです。以下の機能を備えています：

- **ログイン認証**: ユーザー認証機能
- **セッション管理**: gorilla/sessions を使用した安全なセッション管理
- **認証ミドルウェア**: ログイン保護されたページへのアクセス制御
- **HTML テンプレート**: 動的なページレンダリング
- **CORS 対応**: クロスオリジンリクエスト対応のミドルウェア

## プロジェクト構成

```
go-web-sample/
├── main.go                 # メインアプリケーション起動ファイル
├── go.mod                  # Go モジュール定義
├── README.md               # このファイル
├── web/
│   ├── routes.go           # ルート定義
│   ├── auth/
│   │   └── session.go      # セッション管理ロジック
│   ├── handler/
│   │   ├── home.go         # ホームページハンドラー
│   │   ├── login.go        # ログインハンドラー
│   │   └── template.go     # テンプレート処理
│   ├── middleware/
│   │   ├── auth.go         # 認証ミドルウェア
│   │   ├── cors.go         # CORS ミドルウェア
│   │   └── logging.go      # ロギングミドルウェア
│   ├── static/             # 静的ファイル（CSS, JS等）
│   └── templates/
│       ├── home.html       # ホームページテンプレート
│       └── login.html      # ログインページテンプレート
```

## 使用技術

- **言語**: Go 1.23.6
- **依存ライブラリ**: gorilla/sessions (セッション管理)

## セットアップ

### 前提条件

- Go 1.23.6 以上

### インストール

```bash
# リポジトリをクローン
git clone <repository-url>
cd go-web-sample

# 依存ライブラリをインストール
go mod download
```

## 実行方法

```bash
go run main.go
```

サーバーが起動し、`http://localhost:8080` でアクセスできます。

## 主な機能

### ログイン (`/login`)

ログインページを表示します。デモアカウントで認証できます。

### ホームページ (`/`)

ログイン後にアクセス可能なホームページを表示します。ミドルウェアで保護されています。

### ログアウト (`/logout`)

セッションをクリアしてログアウトします。
